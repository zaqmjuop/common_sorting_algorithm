### 统计结果

整数排序最快的是计数排序和基数排序

非整数排序最快的是快速排序

#### 我的看法： 尽量少实现算法，尽量使用开源库或者封装成函数。因为就连这些简单的算法有的甚至都需要遍历递归，遍历递归不是一个很好控制的过程，在Promise内部更加难控制，一个简单的排序甚至都要花费你一天的时间，很不值。
